# Landino's lab macros and code (unordered list)
## Requirements
### For kymograph runs requirement is https://github.com/UU-cellbiology/ZstackDepthColorCode refer how to install plugin of readme section
Note: add update site can also appear as add unlisted site
### For quadrant_vector_analysis /adittion assay csv generator: orientationJ is required: https://bigwww.epfl.ch/demo/orientation/ refer to download section on the left, put .jar on fiji plugins folder

## Run instructions: 
### Change input and output folder to desired: 
Go to plugins > macros > edit and open desired macro
Change $basedir and $outputDir to desired input and output folder absolute paths
